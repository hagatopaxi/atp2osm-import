atp2osm est un outil qui rapproche les points d'intérêt (POI) de la base All The Places avec les objets existants dans OpenStreetMap.

Pour chaque marque, l'outil identifie les correspondances par proximité géographique et par attributs (nom, marque, téléphone, email, site web), puis propose les mises à jour de tags à valider avant import.

Pour voir la progression des imports dans votre région vous pouvez aller visualiser le résultat sur le site suivant : https://nicofredo.github.io/osm_pois_completude_viewer/main.html
